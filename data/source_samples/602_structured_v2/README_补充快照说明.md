# 602期补充结构化快照 V2

## 这是什么

这是基于 `04_补充数据/🧡栋梁材项目数据沉淀.xlsx` 生成的增量补充快照。

它不覆盖 `30_602期结构化数据快照_2026-07-08`。V1 是六表明细事实层；本目录是经营总览、历史对照、成交节奏、LTV 节奏、运营变化解释层。

## 文件

- `supplement_period_overview.csv`：跨营期经营总览和脚本计算口径，2 行。
- `supplement_target_learners.csv`：入营调查和目标学员画像汇总，2 行。
- `supplement_course_question_benchmark.csv`：D1-D10 主课、必修、每日一问补充指标，58 行。
- `supplement_live_benchmark.csv`：D4-D7 直播、订金、补款补充指标长表，110 行。
- `supplement_conversion_cadence.csv`：D4-D11 成交节奏、营业额、LTV 和人群拆分，278 行。
- `supplement_actual_ltv_timeline.csv`：D4-D12 分时累计 LTV 轨迹，81 行。
- `supplement_operations_changes.csv`：运营动作、异常、业务语境解释层，6 行。
- `supplement_traffic_raw.csv`：流量数据沉淀 raw 长表，营期归属未确认，19 行。
- `supplement_reference_links.csv`：常用文件和外部资料链接，4 行。
- `supplement_sparse_raw.csv`：暂未稳定建模的补充 sheet 稀疏原始单元格，519 行。

## 使用边界

- 事实层优先使用 V1 六表快照。
- 本目录里的补充表适合做经营总览、历史对照和解释层。
- `CAC`、`ROI`、广告费用、流量成本等字段需要继续核准来源和覆盖范围，不建议直接对外展示。
- 公式缓存为空的单元格已尽量保留公式或脚本计算值，正式展示前需要复核。
- `supplement_sparse_raw.csv` 只用于保留暂未稳定建模的原始线索，不建议直接进对外看板。
