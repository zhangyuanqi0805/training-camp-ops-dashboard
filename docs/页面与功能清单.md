# 页面与功能清单

## 一级模块

1. 经营总览
2. 营期经营
3. 跨期对比
4. 班级与学员
5. 学习运营
6. 财务效率
7. 数据问答
8. 内部工具

## 关键功能

- 营期切换：601-607。
- 经营时点切换：整期、开营前、D1-D11、截单。
- 首页核心指标：LTV、CAC、营收、学员规模、招生费用。
- 首页每日经营三指标：当日新增 LTV、必修作业率、到课率，支持悬停查看每日精确值。
- 首页总配额占比漏斗：好友、进群、开口、学习、作业、直播、订金、全款。
- 经营链路：成本投入、CAC、学员规模、学习激活、营收、LTV。
- 班级排行：班级营收、人数、LTV、订单。
- 学习运营：主课、每日一问、直播等学习行为。
- 财务效率：营收、LTV、CAC、成本、效率。
- 数据问答：预置经营问题入口。
- 内部工具：数据口径、原始数据、质量检查和更新说明。

## V1.5.3 的主要变化

V1.5.3 增加三条逐日经营曲线，并把总配额占比漏斗改为连续倒梯形。漏斗宽度仍按各层人数除以本营期总配额计算，不表达逐层转化率。
