# 使用说明

## 打开方式

双击包根目录的 `index.html`，再点击「打开经营看板」。

也可以直接打开：

```text
dist/dashboard.html
```

## 页面怎么用

- 左侧一级栏切换大模块：经营总览、营期经营、跨期对比、班级与学员、学习运营、财务效率、数据问答、内部工具。
- 二级侧栏切换当前模块下的细分页面。
- 营期选择器用于切换 601-607。
- Day 切换用于查看整期、开营前、D1-D11 和截单。
- 首页漏斗表示每一层人数占本营期总配额的比例，不表示逐层转化率。

## 同事只看页面时

只需要保留这些文件：

```text
index.html
dist/dashboard.html
docs/
screenshots/
```

但建议完整保留整个文件夹，方便追溯数据来源和后续更新。

## 同事要改数据或继续开发时

需要保留完整文件夹，并查看：

```text
data/runtime/
data/source_samples/
src/
scripts/
qa/
docs/更新数据流程.md
```

## 浏览器建议

优先用 Chrome、Edge 或 Safari 打开。页面是离线 HTML，不需要联网。
